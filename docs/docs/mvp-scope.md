# MVP Scope

The first MVP only tests one thing:

Can gamified progression improve ABAT/RBT learner completion?

## Included

- onboarding flow
- missions
- XP system
- streaks
- AI behavioral scenarios
- readiness scoring
- learner progression

## Not Included

- marketplace
- teletherapy
- employer matching
- payments
- mobile app
- micro-clinics

## Pilot Goal

Run a 4–6 week pilot with:
- psychology students
- BASLP students
- aspiring ABAT/RBT learners

Primary metric:
Certification completion rate.
