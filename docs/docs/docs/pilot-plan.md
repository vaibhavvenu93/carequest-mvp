# Pilot Plan

## Pilot Size

25–50 learners

## Duration

4–6 weeks

## Target Users

- psychology students
- BASLP students
- special education aspirants
- early-stage care professionals

## Primary Metric

Certification completion rate

## Secondary Metrics

- Day 7 retention
- weekly active learners
- missions completed
- simulation participation
- mock readiness score

## Hypothesis

Gamified progression and AI simulations can increase learner consistency and completion compared to passive course systems.
